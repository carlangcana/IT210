  • Hypertext Transfer Protocol (HTTPS)
  • HTTPS or Hypertext Transfer Protocol Secure is currently a widespread internet communication protocol that secures the browsing activity of an individual user and encrypts confidential data he entrusted to a single website on the internet.

  • Differences of HTTP to HTTPS
    • Ports: HTTPS use 443 by default while HTTP use 80
    • Security: HTTPS can prevent the injection of Malwares which intentionally cause malicious attacks. An unprotected HTTP can reveal such informations related to the activity and behavior of the users while using the web. This potential are used by intruders to make advantage to the users by doing a survey of his/her browsing activities regardless if it is personal or private. It is implemented with an extra layer of security (SSL/TLS) which provides privacy and sharing of data between two communicating machines is encrypted.
