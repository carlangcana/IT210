Request Line: The request line is the first line to appear on the header sample followed by an optional request headers. Take this example:

GET /test.html HTTP/1.1
HEAD /query.html HTTP/1.0
POST /index.html HTTP/1.1
Request Headers: The request headers are in the form of name:value pairs. Multiple values, separated by commas, can be specified. Take this example:

Host: www.xyz.com
Connection: Keep-Alive
Accept: image/gif, image/jpeg, */*
Accept-Language: us-en, fr, cn
HTTP message body: Is the data bytes transmitted in an HTTP transaction message.
  • HTTP Requests

    Host: www.test.com
