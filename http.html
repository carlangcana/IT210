  • Hypertext Transfer Protocol (HTTP)
  • The Hypertext Transfer Protocol is the World Wide Web (www) application layer protocol. HTTP is implemented in two programs: a client program and a server program. The client program and server program execute in two different hosts but they talk to each other by exchanging HTTP messages. It is HTP that defines the structure of these messages and how the client and server exchange messages.
  • It works to access web pages or documents from the web servers through browsers and display the results. An object is simply a file, such as a Hypertext Markup Language (HTML) file, a GIF image, a Java applet, an audio clip, etc., that is addressable by a single Uniform Resource Locator (URL). A web page is usually written in HTML file and contains several reference objects. A web client or browser is a user agent for the web and displays the requested information from the web server. The Web server, on the other hand, is the house of the web objects and each has its own addressable URL. When a user defines the URL or clicked a hyperlink, the browser sends request from the web browser to the web server to access web objects.
