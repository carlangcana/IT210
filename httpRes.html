Status Line: The first line is called the status line, followed by optional response header(s). Take this example:

HTTP/1.1 200 OK
HTTP/1.0 404 Not Found
HTTP/1.1 403 Forbidden
Response Headers: The response headers are in the form name: value pairs: Take this example:

Content-Type: text/html
Content-Length: 35
Connection: Keep-Alive
Keep-Alive: timeout=15, max=100
  • HTTP Response
  • After receiving and interpreting a request message, a server responds with an HTTP response message.

    HTTP/1.1 200 OK
    Date: Mon, 3 Sep 2018 01:11:12 GMT
    Server: Apache/1.3.29 (Win32)
    Last-Modified: Sat, 07 Feb 2017
    ETag: "0-23-4024c3a5"
    Accept-Ranges: bytes
    Content-Lengt: 35
    Connection: close
    Content-Type: text/html

    <h1> My Home Page </h1>

    Beige: Status Line
    Pink: Response Headers
